# ATS Ranking System - Task 3

An automated **Applicant Tracking System (ATS)** designed to parse, analyze, and rank candidate resumes against specific job descriptions using Machine Learning.

## 🚀 Project Overview
This system helps streamline the recruitment process by scoring resumes based on keyword relevance, semantic similarity, and skill matching, ensuring the most qualified candidates rise to the top.

## 📁 Repository Structure
* `FUTURE_ML_03.ipynb` - The core Jupyter Notebook containing data preprocessing, feature engineering, model training, and evaluation.
* `README.md` - Project documentation.
